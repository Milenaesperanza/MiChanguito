Supabase URL
Supabase → Settings → API → Project URL

Guía rápida de setup
  1. Creá cuenta en supabase.com (gratis)
  2. Ejecutá el archivo schema.sql en SQL Editor
  3. Creá cuenta en tabscanner.com (gratis, 200/mes)
  4. En Supabase → Edge Functions → Secrets:
    Agregá TABSCANNER_API_KEY con tu clave
  5. Deployá la Edge Function scan-ticket
  6. Pegá URL y Anon Key de Supabase arriba
  7. Subí esta app a netlify.com (drag & drop)
